# AlignEd
Project Description:
AlignEd is an online academic schedule management tool designed specifically for the students of BSINFOTECH at Laguna State Polytechnic University. This website offers a digital space wherein users may be able to arrange, manage, and visualize their class schedules via an interactive weekly timetable display.

This web application gives students the ability to create an account, log in, enter their subjects manually along with their schedule information, and choose subjects for enrollment. All entered information is stored in the MySQL database and fetched dynamically using the PHP REST API backend system.
